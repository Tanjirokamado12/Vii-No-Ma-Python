# Vii no ma

A Wii no Ma Revival...But Now in Python,And More Simple than Node JS version!

## Setup

For Setup,Only thing need is Python and Mobilclip Converter 

Mobilclip Converter:https://github.com/RiiConnect24/mobiclip-conversion-tool/releases

-Install python

-After go to the Directory of Vii no ma

-Run pip install -r requirements.txt

-after run 'Python App.py'

Note:No patcher in Python has been Made,So you need to rely (for now) to Nodejs Patcher at https://github.com/Tanjirokamado12/Vii-no-ma/tree/main/patcher
